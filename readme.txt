je gaat de opdracht niet met readline maken, maar met window.prompt.
Daarmee vraag je de speler om input. Daarmee kun je om iemands naam vragen en ook om nummers.
Met window.prompt krijg je alle antwoorden als string, ook de nummers.
Je kan met parseInt de string "4" omzetten in de integer 4.
Je kan hier wat meer lezen over window.prompt: 
https://javascript.info/alert-prompt-confirm#prompt (edited)
